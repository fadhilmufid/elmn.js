Getting started

Add the runtime, add pages/, and you have a working SPA.

1. App shell

At the site root, use a minimal index.html:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My app</title>
    <script>
      window.ElmnNavigationMode = "pathless";
    </script>
    <script type="module" src="./elmn.js"></script>
  </head>
  <body>
    <div id="elmn"></div>
  </body>
</html>

2. Pages folder

Put routable screens under pages/. Each route is a folder with an index.html. The home page is pages/index.html.

3. Pathless vs URL routing

  • pathless — URL stays on your site root; last screen is restored from localStorage after refresh (mobile-app style).
  • url — normal paths like /todos in the address bar.
GitHub Pages project site — If your app lives at user.github.io/repo-name/, Elmn infers ElmnRoot from the URL so fetches hit /repo-name/pages/... automatically.
