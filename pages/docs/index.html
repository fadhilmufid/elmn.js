Documentation

Elmn is a small client-side router and HTML template runtime. You author pages under pages/, ship static files anywhere, and optionally use pathless “app” navigation.

Quick embed

Use Live Server locally, then include runtime in your shell:

<script type="module" src="https://fadhilmufid.github.io/elmn.js/elmn.js"></script>

Your app shell needs <div id="elmn"></div> in index.html.
