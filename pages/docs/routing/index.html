Routing & pages

Routes mirror the filesystem under pages/—no route config file.

Conventions

  • pages/index.html → /
  • pages/todos/index.html → /todos
  • pages/todos/[id]/index.html → /todos/<anything> (dynamic segment)

Non-route folders

pages/_components/ and pages/components/ are not exposed as URLs. Use them for shared HTML partials loaded with <elmn-component src="/components/...">.

Resolution

The runtime probes for a matching template (static paths first, then dynamic [id] segments). You only need static hosting—no server-side router.
