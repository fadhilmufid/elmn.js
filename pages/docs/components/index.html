Components

Reuse HTML snippets without making them into routes.

Where to put files

Author partials under pages/_components/ (for example nav.html).

Import path

Use a stable virtual path under /components/:

<elmn-component src="/components/nav.html"></elmn-component>

That resolves to pages/_components/nav.html.

Page scripts

Use <elmn-script> in a page for variables and functions, or co-locate index.js next to index.html and export variables, functions, elmnEffect.
